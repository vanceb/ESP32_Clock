# Exporting mask from Kicad

* Kicad PCB -> Export SVG
* Choose the layers to export with the following [options](kicad_svg_export.png)

# Inkscape

* Open the svg file
* Select All
* Path -> Object to path
* Object -> Ungroup
* Object -> Release Clip
* File -> Save As (DXF) with these [options](inkscape_dxf_export_options.png)
